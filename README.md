# 1102-swift

Swift 项目，采用 Swift Package Manager 管理。

## 项目结构

详见 [PROJECT_ARCHITECTURE.md](./PROJECT_ARCHITECTURE.md) 项目架构模块文档。

## 环境要求

- Swift 5.9+
- macOS 13+ / iOS 16+

## 快速开始

```bash
# 克隆项目
git clone git@github.com:doctor-lijy/1102-swift.git
cd 1102-swift

# 构建
swift build

# 运行测试
swift test
```

## 在 Xcode 中打开

```bash
open Package.swift
```
